# Arizbe's - Repostería Artesanal
Sitio web estático para una repostería y panadería artesanal que destaca por su tradición, calidad y sabor de hogar.

Este proyecto fue desarrollado como práctica académica para aplicar estructura semántica en HTML5 y diseño visual con CSS3.

# Descripción
Este proyecto consiste en el sitio web oficial de Arizbe's, una marca ficticia de repostería. El sitio presenta una experiencia visual cálida y acogedora, permitiendo a los usuarios conocer la historia de la marca, explorar el catálogo de productos y ponerse en contacto para pedidos.

El enfoque principal fue la organización del código mediante una estructura de carpetas profesional, el uso de fuentes personalizadas (Google Fonts) y un diseño responsivo básico.

# Tecnologías Utilizadas
HTML5: Estructura semántica avanzada.

CSS: Diseño de layouts con Flexbox y Grid, personalización de fuentes y paleta de colores.

Normalize.css: Para asegurar la consistencia entre distintos navegadores.

JAVASCRIPT: Para funcionalidad de la pagina web

# Requisitos Previos
Navegador web moderno (Google Chrome, Firefox, Edge, Safari).

Editor de código (Recomendado: Visual Studio Code).

# Instalación y Configuración
Clona este repositorio o descarga el archivo ZIP del proyecto.

Asegúrate de mantener la estructura de carpetas (/ASSETS/CSS, /ASSETS/IMG, /PAGES).

Abre el archivo Index.html en tu navegador preferido.

# Uso
Ejecuta el archivo Index.html para entrar a la página de inicio.

Utiliza el menú de navegación superior para moverte entre Inicio, Productos, Nosotros y Contáctanos.

En la sección de Nosotros, conoce la misión y los valores que definen a Arizbe's.

Usa los enlaces del pie de página para acceder directamente a redes sociales o WhatsApp.

# Características
Diseño Coherente: Paleta de colores cálida (Beige, Café y Vinotinto) que evoca la frescura del pan recién horneado.

Tipografía Seleccionada: Uso de fuentes cursivas (Satisfy) para un toque artesanal y fuentes legibles (Bpmf Huninn) para el contenido.

Estructura Semántica: Uso correcto de etiquetas como <header>, <nav>, <main>, <section> y <footer>.

Navegación Intuitiva: Menú consistente en todas las subpáginas mediante rutas relativas organizadas.

Sección de Contacto: Enlace directo a WhatsApp con mensaje predeterminado para facilitar pedidos.
